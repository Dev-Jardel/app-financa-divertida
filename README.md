🐷 Finanças Divertidas - O Gestor Financeiro Gamificado

O Finanças Divertidas é uma aplicação de gestão financeira colaborativa que transforma a tarefa chata de anotar gastos em um "jogo" familiar. O objetivo é incentivar a economia através de rankings, feedbacks bem-humorados e transparência total entre os membros do clã.

✨ Funcionalidades

Sistema de Clãs (Multi-família): Crie uma nova família ou entre em uma existente através de um código único.

Dashboard em Tempo Real: -   Saldo Geral da Casa: A saúde financeira do clã.

Saldo Líquido Individual: Quanto você está contribuindo para a economia.

Ranking da Casa: Quem é o "Mestre das Finanças" e quem é o "Gastador Profissional".

Gamificação e Humor: Feedbacks sarcásticos baseados no seu comportamento (Ex: "Leão do PIX", "Paixão pelo Roxinho").

Histórico Expansível: Controle total sobre as últimas jogadas (transações) da família.

Aprovação de Membros: Segurança garantida com o "Chefe da Família" aprovando novos integrantes.

🚀 Tecnologias Utilizadas

React.js: Biblioteca principal para a interface.

Tailwind CSS: Estilização moderna e responsiva.

Firebase:

Firestore: Banco de dados NoSQL em tempo real.

Authentication: Sistema de login seguro.

Lucide React: Pacote de ícones premium.

Vite: Tooling para um desenvolvimento rápido.

🛠️ Como rodar o projeto localmente

Clone o repositório:

git clone https://github.com/Dev-Jardel/app-financa-divertida.git


Instale as dependências:

npm install


Configure seu Firebase no arquivo App.jsx.

Inicie o servidor de desenvolvimento:

npm run dev
